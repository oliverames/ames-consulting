Stuff I Like

Colophon

This site is hand-written HTML and CSS, hosted on GitHub Pages, with a blog powered by Micro.blog.

Design

The visual identity draws from the Ames Shovel Company — my family's namesake and 225 years of American industrial heritage. Heritage Gold for warmth, Manufacturing Red for emphasis, and condensed industrial typography that says what it means.

Typography is Barlow Condensed for headings and Lora for body text, served from Google Fonts. Light and dark modes follow your system preference.

Technology

  • Hand-authored HTML and CSS (no frameworks, no build step)
  • CSS cascade layers, container queries, Display P3 color
  • Micro.blog for the blog stream
  • GitHub Pages for hosting
  • Claude Code for pair programming

Philosophy

Inspired by lmnt.me — same spirit, different voice. Functional excellence over decoration. Every element earned.
